Realtime enables listening for changes in a PostgreSQL Database

Usecases:

- Cache invalidation
- Forwarding Pub/Sub style subscriptions
- Got another? Open an issue and let me know.


Getting Started
===============

**Requirements**

Python 3.8+

**Installation**
::

    $ pip install realtime
